Browser Default CSS

The first default value the browser sets is actually for the body element's margin. The value most browsers set for the body's margin is 8 pixels on all sides, so all content is pushed away from the top, right, bottom, and left edges of the viewport (browser window).

This is the second paragraph. The browser's default CSS should create a space between this paragraph and the next paragraph.

This is the third paragraph. The space between the second paragraph and this paragraph is based upon the browser's default margin-top and margin-bottom values (about 1em) for paragraphs.
